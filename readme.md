# **Notify Project**

## **專案說明**
<br>藉由Line Notify功能，以撰寫物件導向格式撰寫，連線上傳文字訊息、本地端圖片、網路圖片、Line貼圖。

</br>

## **環境說明**
<br>Windows 10 家用版；
<br>Python version 3.6.8

</br>

## **檔案說明**
<br>notify.py 撰寫內容
<br>image 存放本地端圖片的資料夾
</br>
